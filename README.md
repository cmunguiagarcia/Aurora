# Aurora (Beta)

Aurora is a free macOS utility that replaces the default system volume and brightness OSD with a customizable, iOS-inspired HUD.

Built for users who want a cleaner, more modern on-screen display experience without paying for premium alternatives.

Features

* Custom volume and brightness HUDs
* iOS-inspired design
* Liquid, Frosted, and Solid appearance styles
* Horizontal and vertical HUD layouts
* Independent positioning for volume and brightness HUDs
* Menu bar integration
* Launch at login support
* Native macOS OSD suppression
* Apple Silicon support

Known Issues

* External keyboard brightness controls may bypass Aurora and use the system OSD
* Menu bar text may clip in certain configurations
* Liquid Glass opacity controls are currently non-functional
* Brightness HUD may occasionally fail to appear

Planned Features

* Audio output indicators (AirPods, AirPlay, and more)
* HUD preview button in Settings
* Adjustable animation speed
* Show HUD on the active display
* Optional brightness percentage display

Beta Notice

Aurora is currently in beta and may contain bugs or incomplete features. Feedback, bug reports, and suggestions are greatly appreciated.
